## üèÉ Getting Started

Follow these steps to set up and run the project locally on your machine.

### Prerequisites

Ensure you have the following installed:

* **Node.js** (v18+)
* **npm** or **yarn** (for package management)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/dipti-2211/Dipti-Portfolio.git](https://github.com/dipti-2211/Dipti-Portfolio.git)
    cd Dipti-Portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

### Running the Project

1.  **Start the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

2.  Open your browser and navigate to the local server address (usually `http://localhost:3000`).
